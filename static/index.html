API Documentation

GET /todos

This route will return an array for each todo item in our app. But only the id, is_done, task are included

GET /todos/:id

this route will return a single todo by the todo ID

POST /todos

this route can be used to create a todo item. The body of the request should include the task description

DELETE /todos

this route can delete a todo by the todo's ID

UPDATE /todos

this route update or change a todo by the todo ID, but we cant change the id
